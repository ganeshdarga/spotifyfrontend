For my project, I built a backend using Django REST framework, which provided a powerful and flexible way to create RESTful APIs. I chose Django because of its robust features, including authentication, serialization, and ORM capabilities, which made it easier to interact with my MySQL database.

To make the backend accessible online, I hosted it on Render, using their free tier. Render offers a straightforward deployment process, and their free server option was a great fit for my project at this stage. This setup enabled me to easily deploy my API and make it available to other applications or users.

Note:I hosted my backend on Render's free server, which is cost-effective but comes with some limitations. The server can be slow to respond after periods of inactivity because it goes into "sleep" mode to save resources. This causes a delay in API calls, so when accessing the API, you might need to wait a moment for the data to load. This setup works well for development, though I might upgrade to a more robust solution for better performance in the future.
